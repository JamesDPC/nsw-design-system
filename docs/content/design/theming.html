Theming

The Design System has a built in framework for how colours are used and inherited to ensure a consistent and accessible user experience. The following guidance walks through how each Branding classification can apply colour to its digital products and services using the Design System.

On this page

  • NSW Government colour palette
  • NSW Design System Base colour theme
  • Level 01 colour usage
  • Level 02 and 03 colur usage

NSW Government colour palette

The NSW Government colour palette contains 10 colours sets, plus black and white. The palette is made up of:

  • Colour sets - refers to each column, eg. Green colour set.
  • Tonal sets - refers to each row, 01 being the darkest tonal set and 04 being the lightest tonal set.
  • Dark colours - colours that meet 4.5:1 contrast ratio with Text Light #FFFFFF.
  • Light colours - colours that meet 4.5:1 contrast ratio with Text Dark #22272B.

Colour Usage Levels

There are three colour theming levels in NSW Government:

  • Level 01 - Masterbrand Level 1 and 2
  • Level 02 - Materbrand Level 3 and Co-Brand
  • Level 03 - Independent

For information about NSW Government branding and colour usage classifications please see the [NSW Government brand framework].

NSW Design System Base colour theme

The NSW Design System Base colour theme is based on Level 01 colour usage rules which area applicable for Masterbrand Level 1 and 2 (L1/2).

Brand colours

Level 01 colour usage reflects a strong association to NSW Government through its use of the red, blue and grey colour sets. Core styles and components inherit Brand colours from the base colour theme and their individual guidance will explore the level of customisation or adaptation for each.
Brand Dark

Used for buttons, links, icons, dark variants of components and backgrounds.

Masterbrand L1/2 sites always use Blue 01.
Brand Light

Used for secondary buttons, light variants of components and background.

Masterbrand L1/2 sites can use any blue, grey, or red Light colour.
Brand Supplementary

An optional third colour and can be used for pictograms, component variants and background.

Masterbrand L1/2 sites can use blue, red, or grey from row 02.
Brand Accent

Used in the accent lines found in some components.

Masterbrand L1/2 websites can user any colour from the wider palette.

Level 01 colour usage

The only defined colour for Level 01 is Brand Dark (Blue 01). Brand Light, Supplemetnary and Accent are variable colours and can be altered if required. If you wish to change a variable Brand colour, it must be selected from the Blue, Red or Grey colour sets and:

  • Brand Light must be a Light colour (accessible with Text Dark).
  • Brand Supplementary must be a Dark colour (accessible with Text Light).
  • Brand Accent can be selected from any row in wider colour palette (Row 02 and 03 recommended).

Guidance on applying these in Figma can be found on the Colour page of the Figma UI Kit.

Level 02 and 03 colour usage

Level 02 and 03 colour usage have access to the full colour palette for all Brand colours. This includes Masterbrand Level 3, Co-brand and Independent.

Colour selection rules:

  • Brand Dark must be selected from Row 01.
  • Brand Light must be a Light colour (accessible with Text Dark).
  • Brand Supplementary must be a Dark colour (accessible with Text Light). This is an additional colour option only available if your Brand Light is not from Row 02.
  • Brand Accent can be selected from any row from your chosen colour sets (Row 02 and 03 recommended).

Guidance on applying your new Brand colours in Figma can be found on the Colour page of the Figma UI Kit.

As core styles and components inherit colours from the base colour theme, changing a Brand colour will automatically filter down to all instances where that colour is used. Alternatively, colour instances can be updated on an individual basis as required. Where possible, it is indicated in the guidance on the Components page.
