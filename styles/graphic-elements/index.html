Graphic elements

Injections of the distinctly NSW visual identity.

Graphic elements are part of our visual language that create a consistent user expirence. Like other styles, these are a core part of our brand and should not be altered.

Logo positioning

The logo is positioned in the top left corner to serve as a consistent anchor for users to identify the service or product they are viewing and itâ€™s connection to NSW Government.

This can be achieved using the Header component for Masterbrand or Co-brand.

Border radius

A 4px border radius is applied on clickable containers and inputs, ie:

  • Cards
  • Buttons
  • Form fields
  • Checkboxes

Drop shadow

Drop shadows are used to indicate layering in the interface, ie:

  • Mega menu navigation
  • Predictive list

The "Line System"

The Line system employs Hairlines, Bands and Stripes to:

  • aid information hierarchy
  • support navigational structure
  • segment sections of content
  • create illustrations

The Highlight bar found in various components is an example the of a consistent approach to implementing the line system to aid hierarchy and navigation in digital.

Sections and banners can also be used to apply the line system to segment content.
