Typography

NSW Government uses Public Sans font for all digital products and services to create a distinct but consistent look and feel.

Typeface

Public Sans is the only font to be used by agencies for NSW Government websites and online applications. Major headings through to body copy must also use Public Sans.

Public Sans is a free font and available from Google Fonts.

info

Font migration

The previous iteration of the NSW Design System used Montserrat, to update to Public Sans read through the developer guide here.

System fonts

System fonts are rendered locally by your web browser, this provides greater speed when loading your content. The preferred system font is Arial.

As a good rule of thumb, you should still refer to a system font as a fallback in your CSS font stack:

body {font-family: Montserrat, Arial, sans-serif; }
